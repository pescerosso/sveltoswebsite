Simplifying Kubernetes Add-on and Policy Management

Built for Kubernetes administrators and DevOps teams, Sveltos centralizes and automates add-on and policy deployment across clusters. Scale effortlessly, reduce complexity, and focus on innovation, not configuration.

Key Features of Sveltos

Discover how Sveltos simplifies Kubernetes add-ons management with powerful features designed for modern cloud-native environments.

Centralized Management

Manage multiple Kubernetes clusters from a single control plane, simplifying operations at scale.

Declarative Configuration

Define cluster configurations declaratively, ensuring consistency across environments.

Quick Deployments

Deploy applications effortlessly across clusters using GitOps principles.

Policy Enforcement

Enforce security and compliance policies at scale across your Kubernetes infrastructure.

About Sveltos

What is Sveltos?

  • Comprehensive Solution: Open-source tool for managing Kubernetes clusters efficiently.
  • Kubernetes-Native: Seamlessly integrates into Kubernetes environments using Custom Resource Definitions.
  • GitOps Workflow: Deploy configurations using Git as the single source of truth.

Getting Started

Start Using Sveltos

Follow these simple steps to get started with Sveltos in your Kubernetes environment.

Kubernetes v1.20+ installed and kubectl configured
Run kubectl apply -f https://projectsveltos.github.io/sveltos/addons/addons/
